# Paradise Nursery

## Project Description
Paradise Nursery is a React and Redux-based shopping cart application for browsing and purchasing houseplants. Users can view plants grouped into categories, add items to a shopping cart, update quantities, and view the total cost of selected items.

## Features

### Landing Page
- Displays company name and introduction
- Includes a "Get Started" button that navigates to the product listing page

### Product Listing Page
- Displays houseplants grouped into categories
- Each plant shows image, name, description, and price
- Add to Cart functionality
- Button is disabled once an item is added
- Cart icon displays total number of items in the cart

### Shopping Cart Page
- Displays all selected items
- Each item shows image, name, unit price, and quantity
- Allows increasing and decreasing item quantity
- Allows removing items from cart
- Displays total cost of all items
- Includes Checkout button (placeholder)
- Includes Continue Shopping button

## Technologies Used
- React
- Redux Toolkit
- React Redux
- JavaScript (ES6)
- CSS
